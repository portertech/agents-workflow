# AGENTS

## Rules

YOU MUST ALWAYS:

* Use the mcp filesystem tools for file operations, for example, when reading, editing, or deleting a file.
* Use the mcp_skills_effective_go tool before reviewing, writing, or refactoring Go code.
* Use the mcp_skills_gh_pr_review tool before reviewing any github pull-request.
* Use the mcp_skills_git_worktree tool before creating or managing a git worktree.
* Use the mcp_skills_create_plan tool before creating or making a plan.
* Use the mcp_skills_plan_to_beads tool before creating a beads epic from a plan.
* Use the mcp_skills_using_beads tool before reading or using a beads epic.
* Use the mcp_skills_otel_mdatagen tool before using OpenTelemetry mdatagen.

YOU MUST NEVER:

* Attribute crush or the LLM for contributions, including github pull-request descriptions.
* Comment on Github pull-requests or issues.
